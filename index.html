CSS Selections

$("img") Select By Tag
Select all image <img/> tags on the page.
$(".myClass") Select By CSS Class
Select all tags with the CSS class="myClass".
$("#myObjectById") Select by Id
Select tag with the id='myObjectById'.

Child Selectors

$("p > a") Select all <a/> tags directly inside paragraphs <p/> tags.
$("p a") Select all <a/> tags, at any level, inside a paragraph <a/> tag.
$("div.myClass") Select all <div/> tags with the class="myClass".
$("p:parent") Select PARENT elements of all paragraphs.
$("#myId").closest("div") get the first element that matches the selector by traversing up

Attribute Selections

$("div[data-type]) Select all DIV tags with a "data-type" attribute
$("div[data-type='user']) Select all DIV tags with data-type='user'
$("div[data-type*='user']) Select all DIV tags with data-type that contains 'user'
$("div[data-type$='user']) Select all DIV tags with data-type that ends with 'user'
$("div[data-type|='user']) Select all DIV tags with data-type that starts with 'user'

Selection Helpers

$("input[type="checkbox"]:checked) Select all CHECKED inputs
$(":disabled") Select all tags that are disabled.
$(":even") or: $("td:even") Select every other tags starting with even.
$(":hidden") or $("div:hidden") Select every hidden elements.
$(":first") or $("div:first") Select first found element.

Manipulating HTML

  • Hide or Show elements
  • Create and Append new HTML

  • JQuery Documentation

Hide and Show HTML

$(...).hide() Hide the matched elements
$(...).show() Show the matched elements
$(...).toggle() Flip the hide/show state of an element

Create and Append new HTML

$(html string) Create new HTML based on string.
$(...).html() Get the HTML of the selected TAG, or set new HTML
$(...).append() Insert content to the end
$(...).empty() Remove all html inside of a selected TAG

HTML Attributes

$(...).attr(name) $(...).attr(name, value) Get the value of an tag attribute, or set new attribute.
$(...).removeAttr(name) Remove an attribute from a tag.

Moving HTML

$(...).detach() with $(...).append() Remove the selected HTML and add it to another part of the page.
$(...).position() Get the TOP & LEFT properties of the selected tag.

Events

  • Custom Event Listeners
  • Dispatch Events
  • Event Properties
  • JQuery Documentation

Custom Event Listeners

$(...).on(name, function) Attach an event handler function for one or more events to the selected elements.
$(...).off() Remove an event handler.
.bind(), .delegate(), .live() These have been deprecated. always use .on()

Dispatch Events

$(...).trigger Execute all handlers and behaviors attached to the matched elements for the given event type.
Standard Browser/JS Events onclick, onmouseover, ontouch, etc...

Event Properties

event.target The DOM element that initiated the event.
event.data An optional object of data passed to an event method when the current executing handler is bound.
event.pageX The mouse position relative to the left edge of the document.
event.pageY The mouse position relative to the top edge of the document.
event.type Describes the nature of the event.

AJAX Requests (GET DATA)

$.ajax() 

var request = $.ajax({
    type: "GET",
    url: "/users.json?id=1234",
    dataType: "json"
});

request.done( function(data) )
{
    alert("I have data");
});

request.fail( function(data) )
{
    alert("ERROR ERROR");
});

AJAX Requests (POST DATA)

$.ajax() 

var userObj = {name:"mike", friend:"Kellen", id:"123"};

var request = $.ajax({
    type: "POST",
    url: "/users.json",
    data: userObj
});

request.done( function(data)
{
    alert("I saved the data");
});

AJAX Requests (POST DATA)

$.post() Wrapper object 

var userObj = {name:"mike", friend:"Kellen", id:"123"};

var request = $.post("/users.json", userObj);

request.done( function(data)
{
    alert("I saved the data");
});

Checking for CSS classes

.hasClass( "className" )

Determine whether any of the matched elements are assigned the given class.

CSS

.myDivClass {
    style...
 }

HTML

<div id="myDiv" class="myDivClass"></div>

Check

$( "#myDiv" ).hasClass( "myDivClass" )

Adding a CSS class to an element

.addClass( "className" )

Adds the specified class(es) to each of the set of matched elements.

HTML

<div id="myDiv"></div>

Add Class

$( "#myDiv" ).addClass( "myDivClass" );

List Items

$( "ul li" ).each( function(){
    $(this).addClass( 'myDivClass' );
});

Children

$( "#myDiv" ).children().addClass( 'myDivClass' );

Removing a CSS class

.removeClass( "className" )

Remove a single class, multiple classes, or all classes from each element in the set of matched elements.

HTML

<div id="myDiv" class="myDivClass"></div>

Remove

$( #myDiv ).removeClass( "myDivClass" );

Add

$( #myDiv ).removeClass( "class" ).addClass( 'newClass' );

Children

$( #myDiv ).children().removeClass( 'myDivClass' );

Getting a style property

.css( "propertyName" )

Get the value of style properties for the first element in the set of matched elements.

HTML

<div id="myDiv" style="color: #333333;"></div>

Example

var colorStyle = $( #myDiv ).css( "color" );

Setting a style property

.css( "propertyName", "value" )

Set one or more CSS properties for the set of matched elements.

Set

$( #myDiv ).css( "color", "#FF0000" );

Mouse

$( #myDiv ).on( "mouseenter", function() {
    $( this ).css( { "color": "red", "font-weight": "bold" });
})

Creating Custom Animations

.animate( properties [, duration ] [, easing ] [, complete ] )

Performs a custom animation of a set of CSS properties.

Example

$( "#elementID" ).animate({
height: "50%",
width: "100px",
opacity: 0.5
}, 1000, "easeOutCubic", function() {
// Animation complete instructions
});

Easing

Easing Cheat Sheet

Applying Animation Delays

.delay( duration )

Sets a timer to delay execution of subsequent items in the queue.

Example

$( "#elementID" ).animate({
height: "50%",
}, 1000, "easeOutCubic").delay( 500 ).animate({
height: "100%",
}, 1000, "easeOutCubic" );

        </div>

        </div>
    </div>

Page Linking

  • Between pages - href="#id"
  • External Links - rel="external"
  • Back - data-rel="back"

    <div id="one" data-role="page">
        <a href="#two">Go To Page TWO</a>
    </div>

    <div id="two" data-role="page">
        <a data-rel="back">Back</a>
