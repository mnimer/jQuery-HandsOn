CSS Selections

$("img") Select all images on the page.
$(".myClass") Select all elements with the CSS class="myClass".
$("#myObjectById") Select element with the id='myObjectById'.

Child Selectors

$("p > a") Select all a tags directly inside paragraphs.
$("p a") Select all a tags, at any level, inside a paragraph.
$("div.myClass") Select all elements with the class="myClass" inside of a DIV.
$("p:parent") Select PARENT elements of all paragraphs.

Attribute Selections

$("div[data-type]) Select all DIV elements with a "data-type" attribute
$("div[data-type='user']) Select all DIV elements with data-type='user'
$("div[data-type*='user']) Select all DIV elements with data-type that contains 'user'
$("div[data-type$='user']) Select all DIV elements with data-type that ends with 'user'
$("div[data-type|='user']) Select all DIV elements with data-type that starts with 'user'

Selection Helpers

$("input[type="checkbox"]:checked) Select all CHECKED inputs
$(":disabled") Select all elements that are disabled.
$(":even") or: $("td:even") Select every other element starting with even.
$(":hidden") or $("div:hidden") Select every hidden elements.
$(":first") or $("div:first") Select first found element.
